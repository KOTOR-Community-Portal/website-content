Deadly Stream is the largest and most active host of mods for the KOTOR games. It also contains forums with modding tutorials and ongoing discussions about the games.

### KotOR Patch Manager

The KotOR Patch Manager is a framework which allows programmers to create modular patches which modify the KOTOR games' executable code, unlocking previously hard-coded features.

![GitHub logo](/images/logos/github.svg)

[KotOR Patch Manager](https://github.com/LaneDibello/Kotor-Patch-Manager)
