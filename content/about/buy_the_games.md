  :::note
  Widescreen
  :   Though the Steam version of KOTOR comes with support for the Steam overlay, the executable for the Steam version of the game is encrypted, which prevents patching the game to a resolution higher than the game's native maximum (1280x1024) without replacing the executable file with an editable version. This is relatively easy, and links to modding executables are available safely through DeadlyStream.com (the main KOTOR modding site), but this may still be a consideration for you.

## KOTOR II - The Sith Lords

### PC

KOTOR 2 has a game update unique to the Steam, GOG and Mac App Store versions which adds native controller and widescreen support; this update is referred to as the Aspyr Patch. While its new functionality is helpful, it unfortunately causes several other problems: lightsaber hilt texture mods don't function with the new version, some effects like fog have been removed, save games are sometimes lost temporarily, it can cause or exacerbate issues with in-game dialogue skipping, and rarely users experience extreme and irreparable stuttering and screen tear. Unfortunately, that's just the headline list of issues it creates.
  
You may be thinking to yourself, "well I'll just buy it elsewhere then." Unfortunately it's not that simple; older game versions are *generally* somewhat less stable on modern hardware than the Aspyr-patched game versions are. This instability is generally minor and mostly fixed by [making the executable Large Address Aware](https://ntcore.com/?page_id=371), but this does **NOT** apply to widescreen application. For KOTOR 2, the only stable way to apply widescreen to the game is the Aspyr patch; the Aspyr version offers the only extant support for most resolutions, and even tools like UniWS which do allow for support of certain resolutions like 1920x1080 are seriously unstable even after the executable has been made large address aware. Bottom-line: if you want controller support or widescreen functionality, you should be looking to purchase a copy of KOTOR 2 via a storefront which offers the patched version.
  
Thankfully, through the hard work of the community, on both Steam and GoG (though not the App Store version sadly) almost all of the issues of the Aspyr version can be resolved. Protecting saves is as simple as disabling cloud saves on Steam versions; for everything else, there's the [3C-FD Patcher](https://deadlystream.com/files/file/2734-fog-fix-more-3c-fd-patcher/), an absolute black magic tool which reverts the vast majority of content broken by Aspyr. Two other standalone mods, the [Water Restoration](https://deadlystream.com/files/file/2758-water-restoration/) and [Stutter Fix](https://deadlystream.com/files/file/2699-stutter-fix-and-force-cage-update/) resolve the other two major problems, leaving only two, generally minor, unsolved issues:

1. What can best be described as "dialogue floatiness". Sometimes the dialogue will just... skip, for no good reason. This is different than memory leak-related dialogue skipping, if you're familiar with that in the context of KOTOR. It's more like delayed input reading, where one mouse press to bypass a dialogue line will register as two and instantly skip the next dialogue as well. This is a really annoying bug, but thankfully is fairly rare.

2. When playing with a controller plugged in, even if not actually using the controller, it can cause an issue where the camera becomes vertically offset after upgrading an item in the workbench. Unplugging any controllers while playing resolves the issue entirely, but if you intend to play with a controller the issue must be resolved by saving and loading after it occurs.
